#Omni Downloader Helper Privacy Policy

1. Data Collection: This extension does not collect, store, or transmit any user personal data to external servers.
2. Usage: It only retrieves the URL of the media content currently being viewed by the user and passes it to the local 'Omni Video Downloader' desktop application via a custom protocol handler.
3. Third Party: No user data is shared with third parties.
